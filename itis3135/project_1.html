Project Overview

HTML, CSS, and Javascript will be used to create a webpage for my client.

The purpose of this website is to promote the client's business, House of Leng.

The intended audience are people who are interested in my client's business.

The content of this website will include a homepage, about us page, the menu, and a contact page.

Dynamic Functionality

I plan to include a photo gallery on the homepage and the menu page. If possible, I wanted the images to be interactive, displaying the description of the food. There will also be a form on the contact us page, that will require javascript to submit. I wanted to do something similar to the BYO-introduction where after submitting the form, a different page overwrites the current one.
