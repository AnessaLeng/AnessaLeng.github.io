Website Evaluations

Github

Github is an "AI-powered developer platform to build, scale, and deliver secure software."

  • Contrast: There is contrast on this website. Certain words or sentences stand out.
  • Repitition: There is repitition throughout the homepage. There is an image shown in between blocks of text.
  • Alignment: The text is aligned to the left.
  • Proximity: The proximity between the various blocks of text makes it easier to read the information.

eBay

eBay, an online marketplace where you can buy, sell, and browse various items.

  • Contrast: This website lacks contrast. The background color is white, and the foreground color is black. In addition, the font size is a lot smaller compared to images.
  • Repitition: There is repitition in the layout of the homepage. Images are formatted in a circular shape and font size and color repeat.
  • Alignment: There is alignment on this website. The pictures, titles, and descriptions are aligned.
  • Proximity: The website looks cluttered with items. Although there is a purpose for it, it seems overwhelming.
